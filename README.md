[EN] How to INSTALL ?

Paste this in your SERVER.cfg :
 ensure hud_perfect / start hud_perfect
 
 And modif this HUD/UI with the folder HTML
    
    ⚠️ DON'T TOUCH THE client.lua/fxmanifest.lua/server.lua ⚠️
